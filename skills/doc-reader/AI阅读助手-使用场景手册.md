## 这个 Skill 是干什么的？

**「超级马里奥淘金者」是一个 AI 阅读助手 Skill。** 你把一篇文章（markdown）交给它，它就像超级马里奥顶砖块那样，帮你从文字里敲出闪闪发光的东西，还能把核心概念画成一张可交互的知识图谱。

它由**两件武器**组成：
- 🍄 **超级马里奥淘金者** —— 深挖**一篇文章**的价值
- 🗺️ **知识图谱GPS导航** —— 把概念变成**知识网**，并验证你是否真懂

这两件武器，进一步拆成 **5 个能力维度**（见下一节）——任何阅读任务，本质上都是在组合这 5 维。

**它能帮你解决这些痛点：**

| 你是不是常这样…… | 这个 Skill 怎么治 |
|---|---|
| 读完就忘、抓不住重点 | ① 通读全文，提炼核心观点（金币） |
| 知道文章好，但说不清"好在哪、能干嘛" | ② 提炼 3 个可执行行动（蘑菇） |
| 读了一堆干货，第二天啥也没做 | ② 行动步骤直接落地 |
| 以为自己懂了，一考就懵 | ⑤ 建知识图谱 + 用自己的话讲清楚 |
| 一个主题散落很多文章，拼不出全貌 | ④ 延伸追问 + ⑤ 多篇合并成图 |
| 写作没灵感、对着空白文档卡壳 | ③ 意外发现 + ④ 延伸疑惑 = 写作弹药 |

**一句话定位：** 把"被动读完"变成 **主动淘金 + 建图 + 验证真懂**——读得进、用得上、讲得清。

**怎么用（30 秒）：**
1. 把材料（markdown）放进文件夹（如 `mario-reading-assistant`）
2. 用 Kiro 打开文件夹，选 **Vibe 模式**（Autopilot + Claude）
3. 对话框里 `#引用文件名`，贴入对应 Prompt（淘金者 / 知识图谱GPS导航）

> 💡 Prompt 全文见核心文章中"超级玛丽奥淘金者"和"知识图谱GPS导航"两段提示词。

**🗣️ 触发词（怎么说就能唤起这个 Skill）：**
说任意一句类似的话即可——不必非提"马里奥/淘金者"，有"读 + 挖价值"的意图就会触发：
- "帮我读 / 分析 / 挖掘这篇文章"
- "这篇文章讲了什么"
- "帮我提炼行动点 / 总结一下"
- "做个知识图谱 / 概念关系图"
- 或直接丢文件 + 上面任一句

## 逐维详解

**① 🍄 通读全文（由 AI 与用户共同参与）**
AI 和你一起通读"矿石"，把最核心的观点提炼成要点。👉 这是所有阅读的起点，也是后面四维的地基。
🧰 归属：淘金者 · 第一步

**② 💰 提炼金块，采取行动**
从文章里敲出最有价值的 **3 个可执行步骤**——读完就能动手。👉 专治"读完很激动、第二天全忘"，把抽象内容变成今天就能做的事。
🧰 归属：淘金者 · 第二步

**③ ✨ 意料之外：隐藏的"星星"**
挖出作者没点破、甚至与常识相悖的洞见——超出文本本身的深层见解。👉 这是"淘金"最惊喜的一维，也是写作灵感、爆款角度的富矿。
🧰 归属：淘金者 · 第三步

**④ ❓ 延伸与疑惑：神秘的"问号"，新对话的起点**
抛出 2–4 个最关键的未解之谜，驱动你和 AI 持续追问、扩展到文本之外。👉 对话式学习的发动机——别放过它，挑一个问题继续深挖（见 D2）。
🧰 归属：淘金者 · 第四步

**⑤ 🗺️ 知识图谱GPS导航：知识体系网构建**
把核心概念建成可交互的知识网络（节点-关系-分类），并用"自己的话讲清楚"验证是否真懂。👉 看清全貌 + 检验理解的终极武器，讲不出来就是还没真懂。
🧰 归属：知识图谱GPS导航

> **五维怎么串成闭环？** ① 通读 → ② 行动 → ③ 意外 → ④ 疑惑（持续追问）→ ⑤ 知识图谱GPS导航 建网 + 讲清楚验证 → 回到 ④ 再挖。详见下方 🔁 黄金闭环 一节。

## A. 学习理解

### A1 读论文 / 长技术文章：快速抓住核心
- 😣 **痛点**：论文又长又绕，读完抓不住重点，更不知道能拿来干嘛。
- 🎯 **用哪个**：**淘金者**
- 🔧 **怎么用**：把论文转成 md 扔进矿山 → `#引用` → 贴淘金者 Prompt。重点盯第二步"三大行动步骤"（这篇到底让我做什么）和第三步"意料之外"（作者没点破的洞见）。
- ✨ **进阶**：拿第三步挖到的洞见当起点，顺势进入第四步"延伸疑惑"继续追问 → 自然滑入 D2 对话式探索。

### A2 啃教材 / 专业书章节：把厚书变薄
- 😣 **痛点**：教材信息量大，读完像没读，脑子里一团浆糊。
- 🎯 **用哪个**：**知识图谱GPS导航 为主 + 淘金者辅助**
- 🔧 **怎么用**：先用淘金者通读提炼骨架 → 再用 知识图谱GPS导航 生成该章的知识网络图。**看着图复习**，比啃文字高效十倍。
- ✨ **进阶**：考前直接用 知识图谱GPS导航 做"概念自测"（见 A4）——遮住图，逼自己把节点关系讲一遍，讲不清的就是没掌握。

### A3 学一个全新领域：从 0 建立认知地图
- 😣 **痛点**：面对陌生领域，不知道有哪些核心概念、它们怎么关联。
- 🎯 **用哪个**：**淘金者 + 知识图谱GPS导航（黄金组合）**
- 🔧 **怎么用**：先用 Deep Research 产出领域综述（md） → 淘金者挖出核心观点 + 3 个上手动作 → 知识图谱GPS导航 画出领域概念全景图。一张图你就知道这领域"长什么样"。
- ✨ **进阶**：把图存下来，每学一个新概念就更新它，看着自己的认知地图一点点填满。

### A4 备考概念自测：真懂才算懂
- 😣 **痛点**：以为自己会了，一考就懵——其实是"读懂 ≠ 理解"。
- 🎯 **用哪个**：**知识图谱GPS导航**
- 🔧 **怎么用**：用 知识图谱GPS导航 生成该主题知识图 → 选一个核心节点 → **不看图，用自己的话讲它和谁关联、怎么关联** → 再对照图验证。
- ✨ **进阶**：知识图谱GPS导航 的价值不在"看图"，在"讲清楚"。讲不出来的那条连线，就是你的知识漏洞。

### A5 整理听课 / 讲座笔记
- 😣 **痛点**：笔记记得零散、逻辑散，事后根本不想翻。
- 🎯 **用哪个**：**淘金者**
- 🔧 **怎么用**：把笔记导成 md 扔进矿山 → 淘金者提炼核心观点和行动项。散乱的记录立刻变成结构化要点。
- ✨ **进阶**：多节课的笔记分别淘金后，用 知识图谱GPS导航 合并成一整门课的知识网。

## B. 写作创作

### B1 写作思维枯竭：拿素材当跳板 ✅
- 😣 **痛点**：坐在电脑前脑子一片空白，不知道写什么、怎么切入。
- 🎯 **用哪个**：**淘金者**
- 🔧 **怎么用**：别对着空白文档发呆——**去喂它素材**。把手边相关的文章/资料扔进矿山，淘金者会挖出"行动步骤、意外发现、延伸疑惑"。这三个格子就是现成的写作弹药：
  - **意外发现（星星）** = 新视角、新观点
  - **延伸疑惑（问号）** = 天然选题（读者也好奇）
  - **行动步骤（蘑菇）** = 干货框架
- ✨ **进阶**：先用 知识图谱GPS导航 把你"想写的主题"画成概念图，**想清楚结构再动笔**——先想清，再写顺。

### B2 找选题 / 找爆款角度
- 😣 **痛点**：不知道读者爱看什么，选题干瘪。
- 🎯 **用哪个**：**淘金者**
- 🔧 **怎么用**：批量读行业热门文章，每篇淘金。第四步"延伸疑惑"是最天然的**选题库**（读者也好奇的问题），第三步"意料之外"是**爆款钩子**（反常识才有传播力）。
- ✨ **进阶**：把多篇文章的"意外发现"汇总，挑重叠最多的那一个——往往就是这个领域大家最想被戳破的认知。

### B3 拆解爆款：搞懂它为什么好
- 😣 **痛点**：看到好文章只会说"写得好"，学不到手法。
- 🎯 **用哪个**：**淘金者 + 知识图谱GPS导航**
- 🔧 **怎么用**：淘金者拆它的核心观点怎么立、行动步骤怎么排；知识图谱GPS导航 画出它的论证结构。好文章的"骨架"就暴露出来了。
- ✨ **进阶**：把拆出来的骨架当模板，套到自己的选题上。

### B4 反向润色：自己当读者挖自己的稿
- 😣 **痛点**：自己写的看不出问题，改不动。
- 🎯 **用哪个**：**淘金者**
- 🔧 **怎么用**：把**初稿**扔进矿山，让淘金者当"读者"来挖——它提炼出的核心观点，就是你**真正传达**的东西（常和你以为的不一样）；它的"意外发现/疑惑"能指出哪里没讲透。
- ✨ **进阶**：AI 觉得疑惑的地方，往往就是读者也会卡住的地方，优先补强。

## C. 信息压缩

### C1 啃长报告 / 行业报告
- 😣 **痛点**：几十页报告没时间细读，怕漏掉关键。
- 🎯 **用哪个**：**淘金者**
- 🔧 **怎么用**：报告转 md 扔矿山 → 淘金者提炼核心观点 + 3 个可执行动作。**先吃干货，再按需深入**。
- ✨ **进阶**：想看行业全貌，再用 知识图谱GPS导航 把报告里的概念画成图。

### C2 会议纪要：提炼决议和待办
- 😣 **痛点**：会议记录流水账，会后没人看。
- 🎯 **用哪个**：**淘金者**
- 🔧 **怎么用**：纪要 md 扔矿山 → 重点看第二步"三大行动步骤"，直接变成**待办清单**。
- ✨ **进阶**：把多次会议纪要分别淘金，对比"行动步骤"是否在真正推进。

### C3 政策 / 法规文件：提取对我有影响的条款
- 😣 **痛点**：政策文件术语多、绕，看不出跟自己有什么关系。
- 🎯 **用哪个**：**淘金者**
- 🔧 **怎么用**：文件转 md → 淘金者提炼核心条款和"我该做什么"。知识图谱GPS导航 还能理清条款间的层级关系。
- ✨ **进阶**：⚠️ 涉及法律/合规决策时，淘金者帮你**快速定位重点**，但具体行动建议请找专业人士确认。

### C4 清理收藏夹 / 稍后读
- 😣 **痛点**：收藏了几百篇"以后看"，再也没打开过。
- 🎯 **用哪个**：**淘金者**
- 🔧 **怎么用**：批量处理——每篇淘金，只留第二步"行动步骤"和第三步"意外发现"里值得记的，其余可删。把几百篇**压缩成几十条精华**。
- ✨ **进阶**：同主题的多篇合并淘金，去重后用 知识图谱GPS导航 建主题网。

## D. 深度探索与扩展

### D1 跨文本主题研究 ✅
> 这就是你说的"**扩展用**"的核心——把分散的信息连成体系。
- 😣 **痛点**：一个主题散落在很多文章里，拼不出完整图景。
- 🎯 **用哪个**：**淘金者 + 知识图谱GPS导航（黄金组合）**
- 🔧 **怎么用**：多篇材料**分别淘金** → 把所有挖出的概念喂给 知识图谱GPS导航 → 合并成一张**主题知识网络**。散点自动连成体系。
- ✨ **进阶**：这张主题网就是你研究的"成果地图"，可以直接拿去做汇报 / 写综述。

### D2 对话式探索学习：和 AI 持续追问
- 😣 **痛点**：读完就忘，没有继续深挖的习惯。
- 🎯 **用哪个**：**淘金者（第四步是发动机）**
- 🔧 **怎么用**：淘金者的第四步"延伸疑惑"会给你 2–4 个好问题——**别放过它们**，挑一个继续问 AI，层层深入。这就是核心文章说的"新对话的起点"。
- ✨ **进阶**：随时用 知识图谱GPS导航 把当前讨论的概念可视化，看着图追问，不会跑偏。

### D3 读书会 / 共读：汇总共识
- 😣 **痛点**：共读时各说各的，形不成共识。
- 🎯 **用哪个**：**知识图谱GPS导航**
- 🔧 **怎么用**：每人淘金各自挖到的点 → 汇总喂给 知识图谱GPS导航 → 生成一张大家共享的知识网。**共识和分歧一目了然**。
- ✨ **进阶**：分歧的连线单独拎出来，就是最值得当期讨论的话题。

## F. 知识掌握测试

### F1 备考自测：检验自己是否真懂
- 😣 **痛点**：复习完以为自己会了，上考场发现理解有漏洞——"读懂 ≠ 真懂"。
- 🎯 **用哪个**：**测验模式**
- 🔧 **怎么用**：给测验模式输入[知识点]，按布鲁姆六层次逐层答题。每层即时反馈，六层结束后出总分、错题解析、正确答案参考。**重点盯 ⚠️ 和 ❌ 的层级**。
- ✨ **进阶**：测验暴露的薄弱概念 → 用 GPS 模式建该主题知识图谱 → 可视化概念关系强化理解。

### F2 教学评估：检查学习者掌握程度

## 🔁 黄金闭环（淘金者 + 测验 + 知识图谱GPS导航）

**这是整套工具最强的用法——把六个阶段串成一个会自动转的闭环。**

```
  🍄 淘金者通读挖点          🧪 测验模式逐层出题
  （观点/行动/意外/疑惑）  →  评估掌握程度
        ↑                              ↓
   ❓ 讲不清/薄弱环节           🗺️ 知识图谱GPS导航 把薄弱
    回到淘金/测验追问     ←     概念建成知识网络
```

**五步闭环：**
1. 🍄 **淘金者**通读挖点（观点 / 行动 / 意外 / 疑惑）
2. 🧪 **测验模式**按布鲁姆六层次逐层出题，暴露薄弱环节
3. 🗺️ **知识图谱GPS导航**把薄弱概念建成知识网络
4. 🗣️ 用自己的**大白话**，把核心概念的关系讲一遍
5. ❓ 讲不清的地方 → 回到淘金者第四步追问 → 更新地图 → 回到第 4 步

一直转下去，**理解会越来越深**。

> **一句话总结**：淘金者负责"**输入和深挖**"，测验模式负责"**诊断和评估**"，知识图谱GPS导航 负责"**结构化和验证**"，你负责"**讲清楚**"——讲得出来，才是真懂。
