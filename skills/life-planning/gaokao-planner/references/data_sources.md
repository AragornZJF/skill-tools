# 权威数据来源清单与引用话术

本文件供 gaokao-planner 引用，确保每个关键数字都有可溯来源。

## 一、院校与学科实力

| 来源 | 内容 | 引用话术 |
|------|------|---------|
| 教育部学科评估（第四轮公开 / 第五轮部分公开） | 各高校学科 A+/A/A-/B+ 等档次 | 「根据教育部学科评估，XX大学的XX学科为A类」 |
| 软科中国大学排名 | 综合及学科排名（年度更新） | 「参考20XX年软科中国大学排名」 |
| 双一流建设名单（2017首轮、2022第二轮） | 一流大学/一流学科建设高校与学科 | 「该校XX学科入选第二轮双一流建设名单」 |
| 校友会 / QS / THE（国际排名补充） | 国际视野参考 | 「QS世界大学排名中位于…」 |

## 二、录取分数与位次

| 来源 | 内容 | 引用话术 |
|------|------|---------|
| 各省教育考试院官网 | 一分一段表、投档线、招生计划（最权威） | 「以XX省教育考试院公布的一分一段表为准」 |
| 高校本科招生网/招生章程 | 专业组、选科要求、单科限制、体检要求 | 「详见该校20XX年招生章程」 |
| 阳光高考平台（gaokao.chsi.com.cn） | 教育部官方招生信息平台 | 「参考阳光高考平台公布信息」 |

**重要**：录取线随年份波动，引用时必须标注年份，并提醒「以当年官方公布为准」。位次比分数更稳定，优先用位次定位。

## 三、就业与薪资

| 来源 | 内容 | 引用话术 |
|------|------|---------|
| 麦可思《中国大学生就业报告》（就业蓝皮书） | 各专业就业率、月收入、对口度 | 「据麦可思就业报告，XX专业应届月收入约…」 |
| 各高校毕业生就业质量年度报告 | 该校毕业去向、深造率、薪资 | 「该校就业质量报告显示…」 |
| 国家统计局 / 行业协会报告 | 行业薪酬与人才需求 | 「参考行业薪酬调研数据」 |

## 引用原则

1. 每个录取线/薪资数字都要带【来源 + 年份】。
2. 区分「官方数据」与「参考估算」，后者要明确标注。
3. 数据有不确定性或可能过时时，主动声明截止时间。
4. 永远以各省教育考试院和高校招生章程为最终依据。
