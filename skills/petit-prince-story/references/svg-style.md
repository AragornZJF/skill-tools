# 风格 DNA

## 一句话
《小王子》手绘线稿分镜：纯描边、克制留白、统一骨架、语义贴切。像在白纸上为每个章节画一张安静的故事速写。

## 画风两种（`style` 参数）
### line（默认）
- 全描边 `#141414`，`fill:none`，仅瞳孔 / 实心点填色。
- 透明度分层造层次，背景 `#fbfaf6`。
- **朱砂点睛**：每幕**恰好一个**核心图形用朱砂红 `#C0392B`，其余全黑。即「水墨 + 朱砂 / 画龙点睛」——红只落在该章灵魂所在的那个图形上；金句 / 标题 / 小王子 / 星空 / 沙地 / 内框一律保持黑色。禁止一幕出现 2 处及以上红、禁止把金句或标题涂红。

  点睛部位映射（选哪块点红）：

  | 章节核心 | 朱砂点睛部位 |
  |---|---|
  | 爱 / 独特（玫瑰） | 玫瑰全身（整组描边） |
  | 洞察 / 智慧（狐狸） | 狐狸眼睛 |
  | 生命 / 危险（蛇） | 蛇的信子 |
  | 节奏 / 定时（点灯人） | 灯焰 |
  | 叙述 / 坠落（飞行员） | 机尾 / 螺旋桨 |
  | 旅程 / 终点（王子独行 + 星） | 指路星 / 终点星（造型可变化，见下节） |
  | 并行 / 规模（星群） | 星群中一颗小星球 |
### watercolor
- 几何路径与 line **完全相同**，只加 fill、换描边色。
- 不借上色新增任何元素。

## 寓意文字

每幕底部金句下方加一行寓意说明（`font-size="12"`，`fill="#666666"`），≤12 字纯中文。寓意从原文提炼核心思想，不用整句。

## 指路星造型（王子独行幕专用，每幕择一）
| 造型 | 意境 | 画法 |
|---|---|---|
| 十字星 | 方向 / 启程 | 两线交叉（+），居中实心点，朱砂红 |
| 菱形星 | 探索 / 旅途 | 四角星形，实心填色，朱砂红 |
| 彗星 | 远方 / 召唤 | 星点 + 拖尾细线，朱砂红 |

## 角色区分度规则（相同客人出现多幕时严格执行）

同一角色重复出现时，以下变量必须全换，确保**不靠文字、仅靠视觉就能区分**是哪一幕：
- **星空变量** — 星点位置、十字星坐标每幕重新设计，≤12 颗前提下追求差异
- **小行星变量** — 每幕独立决定小行星数量（0–2 颗）、位置、大小
- **指路星变量（王子独行幕）** — 按意境换造型：十字星、菱形星、彗星尾等，位置随幕变化
- **道具变量** — 可选道具 + 辅助惊喜元素交叉使用，不重复组合
- **氛围变量** — 气候/光效氛围层（星环/光晕/薄雾/影子）每幕择一，不重复使用
- **环境宏观变量** — 月相（新月/满月/无月）、地平线起伏、沙地质感等至少变一处

## 构图惊喜变量（V1–V6，默认关，按需启用）

启用时必须在分镜表中注明语义理由。跨幕不重复使用同一种布局。

## 吸引力检查

每幕生成后自问——这一幕有没有一个"咦？"的瞬间？如果用户扫过时目光不会停留，就重选道具或调布局/氛围，直到有一处让人意外。意外不必大，一个小物件放错位置比多画十颗星有效。连续两幕不允许使用相同的惊喜变体、氛围层或布局编号。

| 编号 | 名称 | 做法 | 适用情境 |
|------|------|------|---------|
| **V1** | **反转** | 客人移至左侧（x≈200–260），王子移至右侧（x≈330–370） | 客人主导叙事 |
| **V2** | **仰角** | 沙地弧上抬 80px，星空间距压缩到上 20%，沙地弧放大到 60% | 渺小 / 仰望 / 孤独 |
| **V3** | **俯角** | 沙地弧下压到 y=720，星空占 80%，星点稀疏 | 宏大 / 哲思 / 天地之间 |
| **V4** | **同框错层** | 客人通过 `translate` 提至半空（y 负值），悬浮在小行星旁 | 梦境 / 回忆 / 非现实 |
| **V5** | **远景** | 王子缩小到 0.35x（`scale(0.35)`），右上角增加第二颗小星球 | 旅程 / 远方 / 探索 |
| **V6** | **偏移** | 王子移到 x=150（偏左）或 x=450（偏右），客人错位但顺序不变 | 失衡 / 错位 / 探索中 |

## 必须
- 纯 SVG，以 `<svg` 开头，W3C 合法，不套 HTML。
- 默认 600×800 竖卡（用户给 size 才改）。
- 描边固定 `#141414`（line）/ 上表深色（watercolor）。
- 大量留白：星点 ≤12，小行星 ≤2。
- 一张分镜只讲一个章节、一个迎客角色、一句金句。
- 每幕最多 1 个辅助惊喜元素 + 1 个氛围层（互不冲突）。
- 启用 V1–V6 时必须在分镜表中注明语义理由。

## 绝对不要
- 不要默认横图。
- 不要 `#1a1a1a` 或随意取色。
- 不要用户没要的道具：心心、对话气泡、法文、枯木、苹果树、标题角标……
- 不要把金句写成整句或多语言。
- 不要每幕重画不同的小王子。
- 不要文字与图形、图形与图形重叠 / 遮挡 / 出框。
- 不要连续两幕使用相同的惊喜元素变体或相同布局。
- 不要在不匹配意境时启用 V1–V6（必须附语义理由）。

## 审美方向
安静、克制、有呼吸感；宁缺毋滥。不要拥挤、花哨、插画化。
