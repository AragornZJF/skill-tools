## Build & Run (from build/ directory)

Build the project:
```
cd build && [build command]
```

Run dev server:
```
cd build && [dev command]
```

## Validation

Run these after implementing to get immediate feedback:

- Tests: `cd build && [test command]`
- Typecheck: `cd build && [typecheck command]`
- Lint: `cd build && [lint command]`

## Operational Notes

All source code and config lives under `build/`. Run all commands from `build/` directory.
